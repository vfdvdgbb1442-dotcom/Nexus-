# واجهة الويب

منصة Nexus - تطبيق الويب التفاعلي

## المتطلبات

- Node.js 16+
- npm أو yarn

## البدء

```bash
npm install
npm run dev
```

## الهيكل

```
src/
├── components/       # المكونات
├── pages/           # الصفحات
├── services/        # الخدمات
├── store/           # إدارة الحالة
├── types/           # أنواع TypeScript
└── App.tsx         # التطبيق الرئيسي
```

## الميزات

- 💬 نظام المحادثات الذكي
- 📊 لوحة التحكم
- 🎨 واجهة حديثة وسهلة الاستخدام
- 📱 تصميم متجاوب (Responsive)
- 🌙 دعم الوضع الليلي
