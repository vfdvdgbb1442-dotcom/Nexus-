# تطبيق الجوال

منصة Nexus - تطبيق الجوال

## التقنيات المستخدمة

- React Native / Flutter
- مشاركة الكود مع تطبيق الويب
- API مركزي موحد

## الميزات

- 📱 تطبيق أصلي للجوال
- 💬 محادثات فورية
- 📊 إدارة المشاريع
- 🔔 إشعارات الفور
- 🌐 عمل بدون اتصال (Offline)

## البدء

```bash
# React Native
npx react-native init NexusMobile

# أو Flutter
flutter create nexus_mobile
```
